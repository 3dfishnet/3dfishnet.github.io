2 Graph Clustering

*⁡maxY~trace(Y~TW1/2KW1/2Y~),(3)\operatorname*{max}_{\tilde{Y}} \mathrm{trace} \left( \tilde{Y}^{T} W^{1/2} K W^{1/2} \tilde{Y} \right), \tag{3} *maxY~​trace(Y~TW1/2KW1/2Y~),(3)

其中， Y~\tilde{Y}Y~ 是列上正交的 n×kn\times kn×k 矩阵并且和权重矩阵 WWW 的平方根成正比，因为 Y~=W1/2Z\tilde Y = W ^{1/2} ZY~=W1/2Z。

3.2 Graph Clustering as Trace Maximization

3.2.1 Ratio association

是最简单的能够转成秩最大化的目标函数。引入向量 xcx_cxc​ ，如果划分 ccc 包含顶点 iii， 那么 sc(i)=1s_c(i)=1sc​(i)=1，否则为 0。于是 ratio association 的目标等价为

*⁡max{∑c=1klinks(Vc,Vc)∣Vc∣=∑c=1kxcTAxcxcTxc=∑c=1kx~cTAx~c},\operatorname*{max} \bigg\{ \sum_{c=1}^{k} \frac{\mathrm{links}(\mathcal{V}_{c}, \mathcal{V}_{c})}{|\mathcal{V}_{c}|} = \sum_{c=1}^{k} \frac{\mathbf{x}_{c}^{T} A \mathbf{x}_{c}}{\mathbf{x}_{c}^{T} \mathbf{x}_{c}} = \sum_{c=1}^{k} \tilde{\mathbf{x}}_{c}^{T} A \tilde{\mathbf{x}}_{c} \bigg\}, *max{c=1∑k​∣Vc​∣links(Vc​,Vc​)​=c=1∑k​xcT​xc​xcT​Axc​​=c=1∑k​x~cT​Ax~c​},

其中 AAA 是图的邻接矩阵， x~c=xc/(xcTxc)1/2\tilde{x}_c=x_c/(x^T_c x_c)^{1/2}x~c​=xc​/(xcT​xc​)1/2，等式之所以成立是因为 xcTxcx_c^T x_cxcT​xc​ 代表划分 ccc 的大小，xcTAxcx_c^T Ax_cxcT​Axc​ 代表 ccc 中所有点的 links\mathrm{links}links 之和。

上式可以写成最大化秩 trace(X~TAX~)\mathrm{trace}(\tilde{X}^T A \tilde{X})trace(X~TAX~)，其中 X~⋅c=x~c\tilde X _{\cdot c}= \tilde{x}_cX~⋅c​=x~c​，并且显然 X~TX~=Ik\tilde{X}^T \tilde{X}=I_kX~TX~=Ik​。容易证明，如果上一节中的 W=IW=IW=I，那么 X~=Y~\tilde{X}=\tilde{Y}X~=Y~。更进一步的，如果 K=AK=AK=A 且 W=IW=IW=I，那么

*⁡maxY~trace(Y~TW1/2KW1/2Y~)=*⁡maxX~trace(X~TAX~).\operatorname*{max}_{\tilde{Y}} \mathrm{trace} \left( \tilde{Y}^{T} W^{1/2} K W^{1/2} \tilde{Y} \right) = \operatorname*{max}_{\tilde{X}} \mathrm{trace} \left( \tilde{X}^{T} A \tilde{X} \right). *maxY~​trace(Y~TW1/2KW1/2Y~)=*maxX~​trace(X~TAX~).

既然这两个秩最大化在数学上是等价的，那我们就可以在邻接矩阵 AAA 上使用加权核k均值，使图的ratio association 单调增加。但是要注意，邻接矩阵 AAA 需要是正定矩阵，这样它是一个有效的核矩阵，可以被分解成 ΦTΦ\Phi ^T \PhiΦTΦ 的形式，从而确保核k均值的目标函数才能在每次迭代时都在下降。（注意正定是收敛的充分而非必要条件。）

3.2.2 Ratio cut

*⁡min∑c=1klinks(Vc,V∖Vc)∣Vc∣.\operatorname*{min} \sum_{c=1}^{k} \frac{\mathrm{links}(\mathcal{V}_{c}, \mathcal{V} \setminus \mathcal{V}_{c})}{|\mathcal{V}_{c}|}. *minc=1∑k​∣Vc​∣links(Vc​,V∖Vc​)​.

定义一个对角矩阵 DDD，Dii=∑j=1nAijD_{ii}=\sum_{j=1}^n A_{ij}Dii​=∑j=1n​Aij​，容易得到

*⁡min{∑c=1kxcT(D−A)xcxcTxc=∑c=1kx~cT(D−A)x~c=∑c=1kx~cTLx~c},\operatorname*{min} \bigg\{ \sum_{c=1}^{k} \frac{\mathbf{x}_{c}^{T} (D - A) \mathbf{x}_{c}}{\mathbf{x}_{c}^{T} \mathbf{x}_{c}} = \sum_{c=1}^{k} \tilde{\mathbf{x}}_{c}^{T} (D - A) \tilde{\mathbf{x}}_{c} = \sum_{c=1}^{k} \tilde{\mathbf{x}}_{c}^{T} L \tilde{\mathbf{x}}_{c} \bigg\}, *min{c=1∑k​xcT​xc​xcT​(D−A)xc​​=c=1∑k​x~cT​(D−A)x~c​=c=1∑k​x~cT​Lx~c​},

其中 L≡D−AL \equiv D - AL≡D−A 是图拉普拉斯矩阵的定义。于是这个转化为最小化 trace(X~TLX~)\mathrm{trace}(\tilde{X}^T L \tilde{X})trace(X~TLX~) 的问题。由于有 trace(X~T(I−L)X~)=trace(X~TX~)−trace(X~TLX~)=k−trace(X~TLX~)\mathrm{trace}(\tilde{X}^T (I-L) \tilde{X})=\mathrm{trace}(\tilde{X}^T \tilde{X}) - \mathrm{trace}(\tilde{X}^T L \tilde{X}) = k - \mathrm{trace}(\tilde{X}^T L \tilde{X})trace(X~T(I−L)X~)=trace(X~TX~)−trace(X~TLX~)=k−trace(X~TLX~)，显然最大化 trace(X~T(I−L)X~)\mathrm{trace}(\tilde{X}^T (I-L) \tilde{X})trace(X~T(I−L)X~) 就等价于最小化 trace(X~TLX~)\mathrm{trace}(\tilde{X}^T L \tilde{X})trace(X~TLX~)，即 Ratio cut的目标。

因此，同上文所述，当 K=I−LK=I-LK=I−L 且 W=IW=IW=I 时，可以将问题转化为核 k-means。注意虽然 LLL 是半正定的，但是 I−LI-LI−L 可能是不定式。

3.2.3 Kernighan-Lin objective

由于要确保每个划分大小相同，也就是要确保 xcx_cxc​ 恰好有 ∣V∣/k|\mathcal{V}|/k∣V∣/k 个非零项。这个约束可以使用 incremental weighted kernel k-means algorithm 解决。

3.2.4 Normalized cut

等价于 normalized association 问题；问题可以表述为

*⁡max{∑c=1k*⁡links(Vc,Vc)*⁡degree(Vc)=∑c=1kxcTAxcxcTDxc=∑c=1kx~cTAx~c},\operatorname*{max} \Bigg\{ \sum_{c=1}^{k} \frac{\operatorname*{links}(\mathcal{V}_{c}, \mathcal{V}_{c})}{\operatorname*{degree}(\mathcal{V}_{c})} = \sum_{c=1}^{k} \frac{\mathbf{x}_{c}^{T} A \mathbf{x}_{c}}{\mathbf{x}_{c}^{T} D \mathbf{x}_{c}} = \sum_{c=1}^{k} \tilde{\mathbf{x}}_{c}^{T} A \tilde{\mathbf{x}}_{c} \Bigg\}, *max{c=1∑k​*degree(Vc​)*links(Vc​,Vc​)​=c=1∑k​xcT​Dxc​xcT​Axc​​=c=1∑k​x~cT​Ax~c​},

其中 x~c=xc/(xcTDxc)1/2\tilde{x}_c=x_c/(x_c^TDx_c)^{1/2}x~c​=xc​/(xcT​Dxc​)1/2。

上式可改写成 $\mathrm{t r a c e} ( Y^{T} D^{-1 / 2} A D^{-1 / 2} Y ) $，其中 Y=D1/2X~Y=D^{1/2} \tilde{X}Y=D1/2X~，并且是正交的。令 (3)(3)(3) 式中 K=D−1AD−1K=D^{-1}AD^{-1}K=D−1AD−1 和 W=DW=DW=D， 可得到 Y=Y~Y=\tilde YY=Y~，从而两个问题等价。如果 KKK 是正定矩阵，即可使用加权k-means迭代最小化 normalized cut。

3.2.5 Weighted graph cuts/association

更一般的，加权的关联问题可以表示成秩最大化问题

*⁡max{∑c=1k*⁡links(Vc,Vc)w(Vc)=∑c=1kxcTAxcxcTWxc=∑c=1kx~cTAx~c},\operatorname*{max} \bigg\{ \sum_{c=1}^{k} \frac{\operatorname*{links}(\mathcal{V}_{c}, \mathcal{V}_{c})}{w(\mathcal{V}_{c})} = \sum_{c=1}^{k} \frac{\mathbf{x}_{c}^{T} A \mathbf{x}_{c}}{\mathbf{x}_{c}^{T} W \mathbf{x}_{c}} = \sum_{c=1}^{k} \tilde{\mathbf{x}}_{c}^{T} A \tilde{\mathbf{x}}_{c} \bigg\}, *max{c=1∑k​w(Vc​)*links(Vc​,Vc​)​=c=1∑k​xcT​Wxc​xcT​Axc​​=c=1∑k​x~cT​Ax~c​},

其中 $\tilde{\mathbf{x}}{c}=\mathbf{x}{c} ( \mathbf{x}{c}^{T} W \mathbf{x}{c} )^{-1 / 2} $。令 Y=W1/2X~Y=W^{1/2}\tilde{X}Y=W1/2X~，可以简化为

WAssoc(G)=*⁡maxY trace(YTW−1/2AW−1/2Y).(4)WAssoc(G) = \operatorname*{max}_{Y} \mathrm{~trace} \Big( Y^{T} W^{-1 / 2} A W^{-1 / 2} Y \Big). \tag{4} WAssoc(G)=*maxY​ trace(YTW−1/2AW−1/2Y).(4)

同样的，加权的划分问题也可以表示成最小化秩的问题

WCut(G)=*⁡min∑c=1kxcT(D−A)xcxcTWxc=*⁡min∑c=1kx~cTLx~c=*⁡minY*⁡trace(YTW−1/2LW−1/2Y).\begin{aligned} WCut(G) & = \operatorname*{min} \sum_{c=1}^{k} \frac{\mathbf{x}_{c}^{T} (D - A) \mathbf{x}_{c}} {\mathbf{x}_{c}^{T} W \mathbf{x}_{c}} \\ & = \operatorname*{min} \sum_{c=1}^{k} \tilde{\mathbf{x}}_{c}^{T} L \tilde{\mathbf{x}}_{c} \\ & = \operatorname*{min}_{Y} \operatorname*{trace} \Bigl( Y^{T} W^{-1 / 2} L W^{-1 / 2} Y \Bigr). \end{aligned} WCut(G)​=*minc=1∑k​xcT​Wxc​xcT​(D−A)xc​​=*minc=1∑k​x~cT​Lx~c​=*minY​*trace(YTW−1/2LW−1/2Y).​

于是 WCut 问题就可以表示成 WAssoc 问题，注意到

trace(YTW−1/2(W−L)W−1/2Y)=k−trace(YTW−1/2LW−1/2Y).\begin{array}{c} \text{trace} \Big( Y^{T} W^{-1 / 2} ( W - L ) W^{-1 / 2} Y \Big) \\ = k - \text{trace} \Big( Y^{T} W^{-1 / 2} L W^{-1 / 2} Y \Big). \end{array} trace(YTW−1/2(W−L)W−1/2Y)=k−trace(YTW−1/2LW−1/2Y).​

因此，在矩阵 W−LW-LW−L 上优化 WAssoc 问题就等价于在 AAA 上优化 WCut 问题。

显然，由于这里的 WWW 也是各个节点权重组成的对角阵，矩阵 YYY 和 3.1 节中的 Y~\tilde YY~ 是相等的。

3.2.6 Summary

令 K=W−1AW−1K=W^{-1}AW^{-1}K=W−1AW−1，于是加权图聚类（划分）问题就可以转化为加权核k-means问题。

同样的，令邻接矩阵 A=WKWA = WKWA=WKW 构造一个图，也可以将加权核k-means转化为加权图聚类问题。
